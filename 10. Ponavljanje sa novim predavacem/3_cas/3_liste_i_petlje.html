3.2 Liste¶

  • Liste se koriste za smestanje niza elemenata u jednu promenljivu

  • Lista je jedan od cetiri ugradjena kontejner tipa u Python-u - tipova koji sluze da drze kolekcije nekih elemenata (postoje jos tuple, set i dictionary)

  • list je rezervisana rec!

In [ ]:
ucenici = ["Pera", "Zika", "Mika"]  # 'ucenici' je lista - kolekcija tri stringa

print(ucenici)
print(f"U ovoj listi ima {len(ucenici)} elementa")
print(type(ucenici))

['Pera', 'Zika', 'Mika']
U ovoj listi ima 3 elementa
<class 'list'>

  • Funkcija len(lista) vraca broj elemenata u listi

  • Elementi u listi su uredjeni, promenljivi i dozvoljeni su duplikati

  • Lista moze sadrzati elemente bilo kog tipa, ne samo primitivne tipove

  • Moze sadrzati i elemente raznih tipova, ali to je pametnije ne koristiti i retko kad zaista ima potrebe za tim u praksi

In [ ]:
lista_intova = [1, 3, 6, 3, 7, 2, 9, 300]

lista_boolova = [True, False, False, True]

mix_lista = ["string", False, 5, 666.666]   # Ne koristiti

3.3 Indeksiranje¶

  • Nacin na koji pristupamo elementima liste je koriscenjem indeksa

  • Indeksi u Python-u pocinju od 0

  • Ovo je uradjeno iz pozadinskih implementacionih razloga brze aritmetike sa memorijskim lokacijama

lista = ['p', 'r', 'o', 'b', 'e']

print(lista[4])

  • Moguce je indeksirati listu i tako da se uzima svaki n-ti element
In [ ]:
lista = [1, 2, 3, 4, 5, 6]

# Ovakvo indeksiranje ide od pocetka do kraja
cela_lista = lista[:]

# Ovakvo indeksiranje uzima svaki drugi clan liste
svaki_drugi = lista[::2]

# Moze i od nazad
unazad = lista[::-1]

print(lista)
print(cela_lista)
print(svaki_drugi)
print(unazad)

[1, 2, 3, 4, 5, 6]
[1, 2, 3, 4, 5, 6]
[1, 3, 5]
[6, 5, 4, 3, 2, 1]

3.4 Slajsovanje¶

  • Cesto je potrebno iz liste izvuci vise podataka odjednom i njih smestiti u zasebnu listu

  • Ovo se radi koriscenjem slicing-a

In [ ]:
meseci = ["Jan", "Feb", "Mar", "Apr", "Maj", "Jun", "Jul", "Avg", "Sep", "Okt", "Nov", "Dec"]

prva_tri_meseca = meseci[0:3]   # Primetite desnu granicu slajsovanja
print(prva_tri_meseca)

poslednji_mesec_u_godini = meseci[-1]
print(poslednji_mesec_u_godini)

['Jan', 'Feb', 'Mar']
Dec
  • Kada se radi slajsovanje, desna granica nije ukljucena! (ovo je kao u matematici poluotvoren skup [prvi, poslednji) )

3.5 Append¶

  • U listu koja je vec formirana moze se dodati novi element koriscenjem funkcije append

  • Ovo je zgodno kad nije unapred poznat broj elemenata koji ce se pojaviti za obradu, vec kako koji pristize, tako se pakuje u listu

In [ ]:
lista = [3, 2, 9, 8]

print(lista)

# Dodaj 5 na kraj liste
lista.append(5)

print(lista)

[3, 2, 9, 8]
[3, 2, 9, 8, 5]

Zadatak¶

  1. Napraviti listu dana u nedelji. Indeksiranjem izdvojiti prvi, cetvrti i sedmi dan i ispisati ih. Slajsovanjem izdvojiti radnu sedmicu i ispisati dane u njoj. Na standardni izlaz ispisati koliko lista ima elemenata i kog su tipa elementi u njoj.

3.6 For petlja¶

  • Petlje (loops) sluze za iterativno ponavljanje bloka instrukcija

  • Naredbe koje iniciraju petlje takodje se smatraju kontrolom toka programa poput if i else naredbi

  • U Python-u dva tipa petlji: for i while

  • For se najcesce koristi za iteriranje po odredjenoj unapred poznatoj sekvenci, a while za iteriranje sve dok se ne ispuni neki uslov

In [ ]:
elementi = [1, 2, 3, 4, 5]

for element in elementi:
    print(element)

  • Cesto se koristi range da se generise neki niz kroz koji se iterira
In [ ]:
for i in range(1, 6):     # Slicno, desna granica nije ukljucena
    print(i)

likovi = ["Raja", "Gaja", "Vlaja"]

for index, lik in enumerate(likovi):
    print(index, lik)

3.8 Continue i break¶

  • Naredbe koje sluze da se iteracija preskoci ili da se prekine prevremeno
In [ ]:
for i in range(1, 11):
    if i == 5 or i == 4:
        continue
    elif i == 7:
        break
    else:
        print(i)
